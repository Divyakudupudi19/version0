# AI Portfolio Website

## About
This project demonstrates how a portfolio website can be created using an AI-powered development tool, **v0 by Vercel**. Instead of writing the frontend code manually from scratch, the website was generated from natural language prompts using AI.

The generated project is built with modern web technologies, including React, Next.js, and Tailwind CSS. The code was downloaded, opened in Visual Studio Code, and managed using Git and GitHub.

## Features
- AI-generated portfolio website
- Responsive and modern user interface
- Built with React, Next.js, and Tailwind CSS
- Easy to customize and extend
- Version controlled with Git and GitHub
